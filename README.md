# Weather

Simple and fast  weather API with Fast API, Celery and Redis 
### Installing

Clone this repository
```
https://github.com/caiaffa/weather-api.git
```

And repeat

```
docker-compose up
```

### Documentation

access
```
http://localhost:5000/docs
```
